Política de Privacidade

Última atualização: Janeiro de 2025

Dados que coletamos

Dados de navegação: Usamos o Google Analytics para entender como o site é usado (páginas visitadas, tempo no site, dispositivo). Esses dados são anônimos.

Cookies: Usamos cookies para:

  • Lembrar suas preferências (moeda favorita, histórico)
  • Exibir anúncios relevantes (Google Adsense)
  • Análise de uso (Google Analytics)

Dados que NÃO coletamos:

  • Não pedimos cadastro
  • Não coletamos nome, e-mail ou dados pessoais
  • Não vendemos dados para terceiros

Seus direitos

Você pode:

  • Desativar cookies no navegador
  • Usar extensões de bloqueio de anúncios
  • Solicitar exclusão de dados via e-mail

Terceiros

Usamos serviços de:

  • Google Analytics (análise)
  • Google Adsense (publicidade)
  • Exchange Rate API (cotações)

Cada um tem sua própria política de privacidade.
